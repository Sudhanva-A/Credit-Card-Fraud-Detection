# Credit Card Fraud Detection App 💳🚨

An end-to-end machine learning app that detects fraudulent credit card transactions using XGBoost, Random Forest, and Logistic Regression.

## 🔍 Features
- Upload transaction features (V1–V28, Time, Amount)
- Switch between ML models
- Real-time stats tracking
- Export predictions as CSV

## ⚙️ Technologies
- Flask + HTML/CSS + JS
- Scikit-learn, XGBoost
- SMOTE for class imbalance
- Deployment ready (AWS/Heroku)

## 🧠 Models Trained
- XGBoost (AUC: 0.9889) ✅
- Random Forest
- Logistic Regression
